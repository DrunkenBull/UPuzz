# BingleStarter

Unity를 이용해서 3 Match 퍼즐 게임을 개발하는 과정을 공유합니다.  
참조 가능한 샘플 수준이 아닌 현재 개발중인 개인 프로젝트의 실제 코드 일부 발췌하여 제작과정을 공유하고자 합니다.

### 구현 목표
* 3x3 기본 매칭 
* 3x3, 4, 5 매칭시 특수블럭 생성
* 장애(젤리, 스톤) 블럭 지원
* 스테이지 파일 지원
* 모바일 해상도 Free 
* 맵/스테이지 선택 등 다양한 UI
* UML을 사용한 설계

### 개발 환경
* OS : Mac 
* 유니티 : Unity 2019.2.10f1 Personal
* 편집기 : Visual Studio for Mac
